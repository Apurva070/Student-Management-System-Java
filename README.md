# 🎓 Student Management System (Java Console Application)

## 📌 Overview

A console-based CRUD application built using Core Java to manage student records.
This project demonstrates strong fundamentals of Object-Oriented Programming (OOP) and file handling.

✔ Designed using **layered architecture (Model–Service–Storage–UI)** for clean and maintainable code.

---

## 🚀 Features

* ➕ Add new student
* 📋 View all students
* 🔍 Search student by ID
* ✏️ Update student details
* ❌ Delete student record
* 💾 Persistent storage using file handling

---

## 🛠️ Tech Stack

* Java (Core Java)
* OOP Concepts (Encapsulation, Abstraction)
* File Handling (Serialization)

---

## 📁 Project Structure

```
StudentManagementSystem/
│── Student.java          # Model class
│── StudentService.java   # Business logic
│── FileHandler.java      # File operations
│── Main.java             # Entry point
```

## ▶️ How to Run

### 1️⃣ Clone the repository

```
git clone https://github.com/Apurva070/student-management-system-java.git
cd student-management-system-java
```

### 2️⃣ Compile the project

```
javac *.java
```

### 3️⃣ Run the application

```
java Main
```
